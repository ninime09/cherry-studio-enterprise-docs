## Subscription Plans

Cherry Studio Enterprise uses a **subscription model** — choose a plan based on your team size:

<img src="/assets/images/en/express/select-plan.webp" alt="Plan selection page" />

Supported payment methods:
- **Alipay** — instant scan-to-pay
- **Bank transfer** — for organizations requiring invoices

## Token Consumption

AI conversations are billed per Token consumed. Different models have different Token unit prices:

<img src="/assets/images/en/admin/provider-management-express-1_en.webp" alt="Token pricing" />

- Top up Token balance at any time
- View org-level and per-member usage in the admin console

## Core Capabilities

| Module | Description |
| --- | --- |
| Core functionality | Multi-model AI chat with multimodal support (text, images, files) |
| Provider management | Built-in mainstream LLM services, out of the box |
| Default model settings | Configure assistant, topic naming, and translation models to balance experience and cost |
| User & group management | Employee accounts and group-based permission control for fine-grained AI resource access |
| Assistant management | Create AI assistants with preset roles, knowledge bases, and tools, distribute by user group |
| Assistant library | Centrally manage reusable assistant templates, one-click import from the built-in template market |
| Agent management | AI agents with multi-model collaboration, skill binding, and MCP tool integration |
| Agent skills | Manage reusable skill modules, add manually or download from the community marketplace |
| Knowledge base | Enterprise RAG engine that vectorizes documents for accurate AI-powered Q&A |
| MCP servers | Connect external tools and data sources via the MCP protocol |
| Mini programs | Integrate external AI tools and web services into the client sidebar |
| Usage analytics | Multi-dimensional dashboards for real-time Token consumption and cost monitoring |
| System settings | Network proxy, search engine integration, OCR document preprocessing, and cache management |
| API gateway | Call enterprise models from external apps and IDE plugins via API Key |

**System support**: Windows 7+, macOS, Linux

For product pricing details, see the official site: https://enterprise.cherry-ai.com/

## Premium (Self-Hosted)

Need full data privacy, integration with private models, or special compliance requirements? Cherry Studio also offers Premium (self-hosted) with a **perpetual license** model, priced by headcount:

| Team Size | Description |
| --- | --- |
| 1–50 users | Starter plan |
| 51–200 users | Mid-sized team |
| 200+ users | Large organization, custom quote |

<Callout type="info">
Premium pricing is customized. [Contact us](/docs/contact) for a detailed proposal.
</Callout>

### Annual Support Fee (Optional)

After purchasing a Premium license, you can optionally add an annual support fee, which includes:

- Version upgrades and new features
- 7x24 technical support
- Remote deployment assistance
- Custom requirement consultation
- Training and usage guidance
