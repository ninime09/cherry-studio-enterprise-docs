## Feature Descriptions

| UI Element | Description | Business Scenario |
| -------- | ----------------------- | ----------------------- |
| **All groups** | Filter by org structure (e.g., Product, R&D) | Evaluate AI ROI by department |
| **All users** | Search and select specific users | Audit abnormal high-frequency calls or token usage |
| **Time range** | Quick select (today, last 7/30 days) or custom | Compare usage before/after a rollout |
| **Refresh** | Reload current data | Get the latest monitoring data |

<Callout type="info">
**Note:** Analytics may have a 1-2 minute delay due to distributed log aggregation. This is normal.
</Callout>
