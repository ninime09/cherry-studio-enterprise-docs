## Dashboard Overview

The core metrics dashboard displays key KPIs as metric cards, with both **Token** and **Cost** views.

<Tabs items={['Token', 'Cost']}>
<Tab value="Token">

<img src="/assets/images/en/admin/usage-analytics-core-metrics-express-token_en.webp" alt="Token Dashboard" />

### Metric Details

| Metric | Description |
| --- | --- |
| **Statistics Requests** | Successful API calls to providers in the selected time range. |
| **Total Token** | Total tokens consumed (input + output). |
| **Input Token** | Prompt and context length sent to AI. |
| **Output Token** | Length of AI responses. |
| **Active Users** | Unique users who made calls in the selected time range. |
| **Active Models** | Number of AI models invoked. |
| **Cache Read Tokens** | Tokens served from cache without recalculation, reducing actual consumption. |
| **Cache Write Tokens** | Tokens written to cache for subsequent reuse. |

</Tab>
<Tab value="Cost">

<img src="/assets/images/en/admin/usage-analytics-core-metrics-express-cost_en.webp" alt="Cost Dashboard" />

### Metric Details

| Metric | Description |
| --- | --- |
| **License Total** | Total monetary quota of the current license. |
| **License Used** | Amount of license quota consumed. |
| **License Remaining** | Unused remaining license quota. |
| **License Requests** | Maximum number of requests allowed by the license. |
| **Statistics Requests** | Actual API calls made in the selected time range. |
| **Statistics Cost** | Actual cost incurred in the selected time range. |
| **Active Users** | Unique users who made calls in the selected time range. |
| **Active Models** | Number of AI models invoked. |

</Tab>
</Tabs>

### Premium Core Metrics Dashboard

<img src="/assets/images/en/admin/usage-analytics-core-metrics_en.webp" alt="Premium Core Metrics Dashboard" />

#### Metric Details

| Metric | Description |
| --- | --- |
| **Total Requests** | Successful API calls to providers. |
| **Total Tokens** | Total tokens consumed (input + output). The most direct cost indicator. |
| **Input Tokens** | Prompt and context length sent to AI. |
| **Output Tokens** | Length of AI responses. |
| **Active Users** | Unique users who made calls in the selected time range. |
| **Active Models** | Number of AI models invoked. |
| **Interrupted** | Requests stopped by users or failed due to network/timeouts. |
| **Avg Tokens/Request** | Indicates average conversation complexity. Higher means more complex or longer context. |

<Callout type="warn">
**Ops tip:** If **Interrupted** spikes, check **System Settings > Model Provider** connectivity or external network stability.
</Callout>

</PremiumOnly>
