## UI Details

* **Search bar:** Search by assistant name or description keywords.
* **List fields:**
  * **Assistant:** Display name and icon.
  * **Group:** The group the assistant is assigned to.
  * **Tags:** Category tags (supports multiple tags).
  * **Description:** Brief summary of the assistant's function.
* **Actions:** Entry points for view, edit, and delete.

## Common Actions

<img src="/assets/images/en/admin/assistant-library-list-management-2_en.webp" alt="" />
Click **Actions** in the top-right to open the menu:

* **Create new assistant template:** Create a new assistant manually.
* **Add from templates:** Import presets from the official Cherry Studio template library.

<Callout type="warn">
**Note:** Deleting assistants is irreversible. It won't affect users who already saved it locally, but new users won't be able to access it.
</Callout>
