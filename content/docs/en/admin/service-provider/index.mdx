## Overview

Cherry Studio Enterprise (Express) gives individuals and teams a zero-configuration AI workspace. No need to register ChatGPT or Claude accounts, apply for overseas credit cards, or configure special networks — use the world's top models directly.

## Core Features

* **Out of the box**: No API keys or proxy setup required.
* **Top global models**: Built-in **GPT-5**, **Claude 4.5 Sonnet**, **Google Gemini 3 Pro**, and more.
* **Official pricing**: Billing matches upstream providers (OpenAI / Anthropic) exactly. **No markup**. Pay as you go; balance never expires.
* **Unified billing**: One account balance covers all models — no need to juggle multiple subscriptions like ChatGPT Plus and Claude Pro.

## Ideal For

* Users who cannot apply for overseas API keys.
* Writers who want to start immediately without dealing with network setup.
* Users who need premium models (e.g. Claude 4.5) occasionally without monthly subscriptions.
