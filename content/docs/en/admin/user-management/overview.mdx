## Core Areas <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-he-xin-gong-neng-qu-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-he-xin-gong-neng-qu-0"></a>

**1. Global Search and Filters**

The toolbar at the top provides efficient search:

* **Search users**: Fuzzy match by **username** to quickly locate employees.
* **All statuses**:
  * **Active**: Accounts in normal use.
  * **Inactive**: Disabled or not yet initialized accounts.
* **All groups**: Filter by user group (e.g., "Default", "Engineering") for department-based permission management.

**2. User List Details**

The table shows key identity info:
