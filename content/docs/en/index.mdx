## Product Overview

Cherry Studio Enterprise is a cloud-hosted AI workspace for teams — no deployment, sign up and go:

- **Zero deployment**: ready immediately after sign-up; no servers, no infrastructure to set up
- **Subscription-based**: choose a plan that fits your team size; supports Alipay and bank transfer
- **Built-in models**: OpenAI, Claude, Gemini and more pre-configured — employees don't have to wire up anything

## Data Security and Isolation

Cherry Studio Enterprise provides multiple layers of data protection:

| Security | Description |
| --- | --- |
| **Data Isolation** | Strict group-based isolation — each organization's data is completely separate and invisible to others |
| **Encryption in Transit** | End-to-end HTTPS encryption |
| **Authentication** | JWT + OAuth2 with encrypted password storage |
| **Access Control** | Role-based fine-grained permissions (Admin / Group Admin / Member) |
| **API Security** | Independent API keys with per-user access isolation |

<Callout type="info" title="Data Isolation">
Cherry Studio Enterprise uses shared infrastructure with application-level isolation. Each organization's data (model configurations, knowledge bases, assistants, files, etc.) is strictly isolated by group. Every API request goes through authentication and permission checks, ensuring access is limited to your organization's data only.
</Callout>
