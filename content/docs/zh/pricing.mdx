## 核心能力

| 功能模块 | 说明 |
| --- | --- |
| 基础功能 | 多模型 AI 对话、多模态支持（文本、图像、文件） |
| 服务商管理 | 内置国内外主流大模型服务，开箱即用 |
| 默认模型设置 | 配置助手模型、话题命名和翻译模型，平衡体验与成本 |
| 用户与分组管理 | 员工账号管理与分组权限控制，精细分配 AI 资源访问权限 |
| 助手管理 | 创建预置角色、知识库和工具的 AI 助手，按用户组分发 |
| 助手库 | 集中管理可复用的助手模板，支持从内置模板市场一键导入 |
| 智能体管理 | 支持多模型协作、技能绑定和 MCP 工具集成的 AI 智能体 |
| 智能体技能 | 管理可复用的技能模块，支持手动添加或从社区市场下载 |
| 知识库管理 | 企业级 RAG 引擎，将文档向量化实现精准问答 |
| MCP 服务器 | 通过 MCP 协议连接外部工具和数据源 |
| 小程序管理 | 将外部 AI 工具和 Web 服务集成到客户端侧边栏 |
| 用量统计 | 多维度可视化看板，实时监控 Token 消耗与成本分布 |
| 系统设置 | 网络代理、搜索引擎集成、OCR 文档预处理和缓存管理 |
| API 网关 | 支持通过 API Key 在外部应用、IDE 插件中调用企业模型 |

**系统支持**：Windows 7+、macOS、Linux

产品定价信息参考官网：https://enterprise.cherry-ai.com/

## Premium 版（私有化部署）

需要数据完全私有、对接私有化模型，或有特殊合规要求？Cherry Studio 还提供 Premium 版（私有化部署），采用**授权许可证**（买断制），按用户人数阶梯定价：

| 人数规模 | 说明 |
| --- | --- |
| 1-50 人 | 起步方案 |
| 51-200 人 | 中型团队 |
| 200 人以上 | 大型组织，定制报价 |

<Callout type="info">
Premium 版为定制报价，请 <DynamicLink href="/[lang]/docs/contact">联系我们</DynamicLink> 获取详细方案。
</Callout>
