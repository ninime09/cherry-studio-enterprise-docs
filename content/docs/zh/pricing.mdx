## 版本选择指南

### Express 版 vs Premium 版

两个版本在**核心功能上完全一致**，都包含完整的企业级能力。主要区别在于部署方式和计费模式：

| | **Express 版**（云端托管） | **Premium 版**（私有化部署） |
| --- | --- | --- |
| **部署方式** | 无需部署，注册即用 | Docker / Helm 自行部署 |
| **数据存储** | Cherry Studio 云端托管 | 完全在您自己的服务器上 |
| **服务商配置** | 内置国内外主流大模型，开箱即用 | 完全自由配置，支持自定义服务商和私有化模型 |
| **计费方式** | 按套餐订阅 + Token 按量消耗 | 授权许可证（买断制） |
| **用户人数** | 按套餐阶梯 | 按许可证阶梯 |
| **技术支持** | 在线客服 | 专属技术支持（需购买年度服务费） |

### 如何选择？

- **选择 Express 版**：希望零部署、即刻上手，使用主流大模型服务，按需付费
- **选择 Premium 版**：需要数据完全私有、对接私有化模型、或有合规要求

## Premium 版定价

Premium 版采用**授权许可证**（买断制），按用户人数阶梯定价：

| 人数规模 | 说明 |
| --- | --- |
| 1-50 人 | 起步方案 |
| 51-200 人 | 中型团队 |
| 200 人以上 | 大型组织，定制报价 |

<Callout type="info">
Premium 版为定制报价，请 <DynamicLink href="/[lang]/docs/contact">联系我们</DynamicLink> 获取详细方案。
</Callout>

### 年度服务费（可选）

购买 Premium 版许可证后，可选购年度服务费，包含：

- 版本升级和新功能更新
- 7x24 小时技术支持
- 远程部署协助
- 定制化需求沟通
- 培训和使用指导
