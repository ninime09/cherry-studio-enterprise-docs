## 概述

<Callout type="info">
**入口路径：** 管理后台 → 服务商管理
</Callout>

<img src="/assets/images/zh/admin/provider-management-standard-1_zh.webp" alt="" />
在此模块中，管理员可以统一配置和管理企业所使用的 AI 模型来源。Cherry Studio 预置了主流大模型厂商的连接协议，支持一键接入 OpenAI、Anthropic (Claude)、Google (Gemini) 以及各类兼容 OpenAI 协议的国内中转服务商（如 SiliconFlow、DeepSeek、OpenRouter 等）。

<Callout type="info">
**旗舰版定义的范围**：指通过标准的 API Key 和 API Host 方式接入的商业化模型服务。
</Callout>

## 添加新服务商

**操作：** 点击列表页底部的 **+ 添加服务商** 按钮。

<img src="/assets/images/zh/admin/provider-management-standard-2_zh.webp" alt="" />
系统支持通过预设模板快速接入主流厂商。

## 模型列表与能力标签

添加服务商后，点击该服务商卡片即可展开模型列表。系统会自动尝试拉取该服务商支持的模型列表，您也可以手动添加。

## 模型参数与计费配置

**操作：** 点击具体模型右侧的 **⚙️ 设置** 图标。

为了精确统计企业运营成本，建议管理员详细配置每个模型的计费规则。

<img src="/assets/images/zh/admin/provider-management-standard-4_zh.webp" alt="" />
### 编辑模型设置

* **模型 ID**：API 调用时的真实字符串（不可随意修改，必须与厂商文档一致）。
* **上下文长度**：限制该模型的最大记忆窗口（如 `32k`, `128k`）。
* **输入/输出价格**：
  * **输入价格**：每百万 Token 的提问费用。
  * **输出价格**：每百万 Token 的回答费用。
  * **货币类型**：支持 `CNY` (人民币) 或 `USD` (美元)。

<Callout type="warn">
**为什么配置价格很重要？** 在此处输入的单价未来将直接关联到 **\[用量统计]** 模块。如果留空，后台将无法计算该模型的消耗成本，导致财务报表数据缺失。
</Callout>

## 连通性验证

配置完成后，为确保 API Key 有效且网络通畅：

<img src="/assets/images/zh/admin/provider-management-standard-5_zh.webp" alt="" />
1. 点击服务商配置栏右侧的 **检查连接** 按钮。
2. 在弹出的对话框中选择一个具体的模型（如 `deepseek-chat`）。
3. 点击 **确认**。
4. **成功反馈**：右下角弹出提示 "检查成功 API Key 有效"。

## 常见问题 (FAQ)

**Q: 添加了服务商但模型列表是空的？**&#x20;

A: 部分兼容 OpenAI 协议的服务商不支持自动列出模型 (`/v1/models` 端点)。此时您需要点击列表底部的 **+ 添加模型**，手动输入模型 ID（如 `qwen-max`）。

---

**Q: 为什么我的助手无法使用知识库？**&#x20;

A: 请检查该助手使用的模型是否带有 🔧 (工具调用) 图标。部分轻量级模型或旧版模型不支持工具调用，无法触发知识库检索。
