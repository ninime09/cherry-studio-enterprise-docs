## 编辑用户信息 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-bian-ji-yong-hu-xin-xi-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-bian-ji-yong-hu-xin-xi-0"></a>

若需修改用户的分组归属或账号状态：

1. 在用户列表中找到目标用户。
2. 点击右侧 **操作** 列的 **编辑** 按钮。
3. 在弹出的 **编辑用户** 面板中进行修改：
   * **权限变更**：修改 **所属分组**，即刻生效。
   * **账号封禁**：关闭 **激活状态** 开关，该用户将立即失去登录权限。
   * **提升管理员**：开启 **组管理员** 开关，将该用户设为管理员。
4. 点击 **更新用户** 保存。

编辑用户面板包含：所属分组、用户名、激活状态、组管理员。

<img src="/assets/images/zh/admin/user-management-edit-user-modal-express_zh.webp" alt="编辑用户弹窗" />

<Callout type="info">
用户通过邀请链接自行注册，密码由用户自行管理，管理员无法查看或重置密码。
</Callout>

## 删除用户 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-shan-chu-yong-hu-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-shan-chu-yong-hu-0"></a>

对于离职员工或废弃的测试账号：

1. 点击 **操作** 列的 **⋮** 图标。
2. 选择 **删除用户**（红色选项）。

<img src="/assets/images/zh/admin/user-management-edit-user-1_zh.webp" alt="" />
3. 系统将弹出二次确认提示，确认无误后执行删除。

<img src="/assets/images/zh/admin/user-management-edit-user-2_zh.webp" alt="" />
<Callout type="error">
**Warning:** 删除操作是**不可逆**的。删除后，该用户关联的个人知识库配置、历史对话记录可能无法恢复，请谨慎操作。
</Callout>
