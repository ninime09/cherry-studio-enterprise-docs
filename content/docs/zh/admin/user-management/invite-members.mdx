## 功能说明

成员邀请允许管理员生成专属邀请链接，团队成员点击链接后自行注册并加入组织，无需管理员逐一手动创建账号。

适用于：
- 团队快速扩容
- 新员工自助入职
- 跨部门协作人员加入

## 生成邀请链接

### 操作步骤

1. 进入管理后台 → 用户管理
2. 点击「邀请成员」按钮
3. 配置邀请选项（见下方参数说明）
4. 复制生成的邀请链接

<img src="/assets/images/zh/express/invite-members.webp" alt="邀请链接生成界面" />

### 邀请配置参数

| 参数 | 说明 |
| --- | --- |
| **目标分组** | 被邀请人加入后所属的分组，决定其可访问的模型和助手资源 |
| **有效期** | 邀请链接的有效时间（如 7 天、30 天或永不过期） |
| **使用次数** | 邀请链接的最大使用次数（如不限制、10 次、50 次） |

## 管理邀请链接

邀请链接在分组设置中进行管理。您可以为每个分组创建独立的邀请链接，并控制有效期和使用次数。

详见 <DynamicLink href="/[lang]/docs/admin/group-management/invite-links">邀请链接管理</DynamicLink>。

## 被邀请人注册流程

被邀请人收到链接后：

1. 点击邀请链接，进入「加入组织」页面
2. 页面自动显示邀请码和即将加入的分组信息
3. 确认信息无误后，点击「立即加入」完成加入
4. 下载客户端并登录即可使用

<Callout type="info">
被邀请人需先注册账号并登录后，才能通过邀请链接加入组织。客户端登录方法详见 <DynamicLink href="/[lang]/docs/client">客户端</DynamicLink>。
</Callout>

<img src="/assets/images/zh/express/invite-join.webp" alt="被邀请人注册页面" />

## 注意事项

- 邀请链接应妥善保管，避免发送给非目标人员
- 建议为邀请链接设置合理的有效期和使用次数限制
- 被邀请人注册后，管理员仍可在用户管理中编辑其分组和权限
- 如需撤销已注册用户的访问权限，可在用户管理中将其设为"未激活"状态
