## 配额层级

Cherry Studio 采用两级配额体系：

1. **分组级配额**：在分组设置中配置，作为该组所有成员的默认限额。
2. **用户级配额**：针对单个用户的个性化配额，优先级高于分组配额。留空则继承分组默认值。

## 为用户设置配额

1. 前往 **用户管理** 页面。
2. 在目标用户的操作列中，点击 **限额** 按钮。

<Tabs items={['Express 版', 'Premium 版']}>
<Tab value="Express 版">

<img src="/assets/images/zh/admin/user-management-quota-dialog-express_zh.webp" alt="Express 版用户配额设置" />

Express 版支持通过顶部的 **金额限额** / **Token 限额** 标签页切换，分别设置金额维度和 Token 维度的配额。

</Tab>
<Tab value="Premium 版">

<img src="/assets/images/zh/admin/user-management-quota-dialog_zh.webp" alt="Premium 版用户配额设置" />

</Tab>
</Tabs>

3. 在弹窗中设置各维度的限额值：

<table><thead><tr><th width="100">维度</th><th>说明</th></tr></thead><tbody><tr><td><strong>日限额</strong></td><td>过去 24 小时内允许的最大使用量。每日自动重置。</td></tr><tr><td><strong>周限额</strong></td><td>过去 7 天内允许的最大使用量。每周自动重置。</td></tr><tr><td><strong>月限额</strong></td><td>过去 30 天内允许的最大使用量。每月自动重置。</td></tr><tr><td><strong>总限额</strong></td><td>累计总使用量上限。不自动重置，需手动重置。</td></tr></tbody></table>

<Callout type="info">
**提示：** 留空表示继承分组默认配置。如果分组也未设置，则表示不限制。点击弹窗左下角的 **恢复默认** 按钮可一键清空该用户的所有自定义限额，回到继承分组默认值的状态。
</Callout>

## 查看用量与重置

点击每个限额维度旁的展开按钮，可以查看当期用量详情：

<Tabs items={['Express 版', 'Premium 版']}>
<Tab value="Express 版">

<img src="/assets/images/zh/admin/user-management-quota-detail-express_zh.webp" alt="Express 版配额用量详情" />

* **当期用量**：以 `已用量 / 限额` 格式显示当前周期的使用情况。
* **过去 24 小时**：最近 24 小时内的消耗金额。
* **下次重置**：该维度配额自动重置的时间。
* **重置用量**按钮：手动将当期用量清零。

</Tab>
<Tab value="Premium 版">

<img src="/assets/images/zh/admin/user-management-quota-detail_zh.webp" alt="Premium 版配额用量详情" />

* **当期用量**：当前周期内已使用的量。
* **限制值**：当前设置的上限（"无限制"表示未设置）。
* **下次重置时间**：该维度配额自动重置的时间。
* **重置用量**按钮：手动将当期用量清零。

</Tab>
</Tabs>

4. 配置完成后点击 **保存**。

<Callout type="warn">
**注意：** 当用户使用量达到限额时，后续的 API 调用将被拒绝，直到配额重置或管理员手动调整。
</Callout>

## 临时额度调整

<Callout type="info">
**适用版本：** 当前在 Express 云端版可用。本次追加的额度在该周期重置后自动失效，不修改原始限额配置。
</Callout>

当某位用户因紧急需求需要突破现有配额（例如月限额已满但仍需再跑一批任务）时，您可以使用 **临时加额** 为该用户的当期周期注入额外额度，无需修改其常规配额设置。

### 临时加额

1. 进入 **用户管理**，在目标用户操作列点击 **限额** 打开用户限额弹窗。
2. 切到 **金额限额** 或 **Token 限额** 标签页。
3. 在 **日限额 / 周限额 / 月限额** 字段右侧（图表按钮旁），点击 **+** 按钮。
4. 在弹出的 **「日/周/月 临时加额」** Popover 中输入要追加的数值（正整数，单位随所在 Tab — Token 数或货币金额），点击 **应用** 提交。

<img src="/assets/images/zh/admin/user-management-quota-adjust-popover_zh.webp" alt="用户限额 - 临时加额" />

<Callout type="warn">
临时追加的额度仅作用于**当前周期**，周期自动重置时会随当期用量一起清零；不会修改用户的常规限额值。**总限额** 字段不支持临时加额。
</Callout>

### 查看调额历史

在 **用户管理** 页面，点击目标用户右侧的 **⋯** 操作菜单 → 选择 **调额记录**。

<img src="/assets/images/zh/admin/user-management-quota-adjust-entry_zh.webp" alt="调额记录入口" />

调额记录弹窗按时间倒序展示该用户跨所有周期的全部历史调整：

| 列 | 说明 |
|---|---|
| **时间** | 调整发生的时间戳 |
| **周期** | 该次调整作用的周期（日/周/月） |
| **单位** | 调整的额度单位（金额 / Token） |
| **增量** | 本次追加的数值 |
| **操作人** | 执行调整的管理员账号 |

<img src="/assets/images/zh/admin/user-management-quota-adjust-history_zh.webp" alt="用户调额记录" />
