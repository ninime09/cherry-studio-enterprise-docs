## 核心功能区 <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-he-xin-gong-neng-qu-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-he-xin-gong-neng-qu-0"></a>

**1. 全局搜索与筛选**

位于页面顶部的工具栏提供高效的查找功能：

* **搜索用户**：支持输入**用户名**进行模糊匹配，快速定位特定员工。
* **全部状态**：
  * **活跃**：正常使用的账号。
  * **未激活**：已被禁用或尚未完成初始化的账号。
* **全部分组**：根据用户的所属分组（如"默认分组"、"开发组"）进行筛选，便于按部门管理权限。

**2. 用户列表详情**

数据表格直观展示了用户的关键身份信息：
