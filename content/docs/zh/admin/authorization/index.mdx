<Callout type="info" title="仅 Premium 版">
授权管理（许可证）适用于 Premium 版（私有化部署）。Express 版通过订阅制计费，无需手动管理许可证。
</Callout>

**授权管理** 是 Cherry Studio 企业版的运行核心。作为系统管理员，您需要在此模块中监控系统的合法使用状态、管理用户配额，并处理许可证的更新与迁移。

该模块确保您的企业 AI 服务在合规、稳定的环境下运行，同时防止因授权过期导致的服务中断。

</PremiumOnly>
