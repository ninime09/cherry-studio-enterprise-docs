**当前 License 信息** 面板展示了许可证的具体规格。理解这些参数有助于您进行容量规划。

{/* TODO: Add screenshot for authorization parameters */}

| 参数字段           | 描述说明                              | 关键作用                                                 |
| -------------- | --------------------------------- | ---------------------------------------------------- |
| **License ID** | 唯一的授权标识字符串 (UUID)。                | **售后凭证**。在联系 Cherry Studio 技术支持时，必须提供此 ID 以定位您的企业账户。 |
| **激活时间**       | License 首次被导入并激活的日期。              | 记录系统的上线时间点。                                          |
| **过期时间**       | License 失效的截止日期。                  | **续费提醒**。系统通常会在过期前提供倒计时提醒，请务必在此日期前完成续费。              |
| **状态**         | 当前法律效力 (如：`有效` / `已过期`)。          | 决定系统是否可用。                                            |
| **公司名称**       | 获得授权的企业实体名称。                      | 确保授权主体与您的企业一致，用于合规审计。                                |
| **用户限制**       | **已创建用户数 / 最大授权用户数** (如 `6/200`)。 | **容量控制**。当已创建用户数达到上限 (200) 时，**用户管理** 页面将禁止创建新用户。    |
| **授权功能**       | 当前版本包含的高级特性包 (如 `express`)。       | 标识您购买的产品等级（如标准版、企业版、旗舰版）。                            |

<Callout type="warn">
**容量预警：** 当 **用户限制** 进度条接近 90% 时，建议立即联系销售团队进行扩容，以免新员工无法入职使用。
</Callout>
