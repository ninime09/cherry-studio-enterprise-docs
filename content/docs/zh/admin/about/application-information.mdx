**系统更新**（仅 Premium 版）：配置软件更新后，可检测并升级到最新版本。需先点击"配置软件更新"完成配置；配置完成后可在线检测和安装更新。

<Callout type="warn">
**注意：** 如果 **服务状态** 显示异常，虽然前端页面可能仍能访问，但模型调用、文件解析等核心后端功能将会失效。
</Callout>

## 资源下载中心

管理员可在此处快速获取配套工具，确保客户端与服务端的版本匹配：

* **客户端下载：** 提供 Windows 和 macOS 版本的 Cherry Studio Enterprise 客户端下载链接。
* **帮助文档：** 跳转至官方详细的操作手册（即您当前阅读的文档库）。

<PremiumOnly>
* **Docker 镜像** *(仅 Premium)*：跳转至 Docker Hub 官方仓库，用于服务器端的升级维护操作。

<Callout type="info">
**国内镜像地址** *(仅 Premium)*：中国大陆用户建议使用国内镜像，拉取速度更快、更稳定：
```
docker-registry.cherry-ai.com/public/cherry-studio-enterprise-api:v0.4.10
```
</Callout>
</PremiumOnly>
