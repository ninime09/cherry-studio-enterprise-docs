## 界面功能详解

* **搜索栏：** 支持通过助手名称或描述关键词快速查找。
* **列表字段：**
  * **助手：** 显示名称及图标。
  * **所属分组：** 显示该助手被分配到的分组。
  * **角色：** 显示该助手所属的分类标签（支持多标签）。
  * **描述：** 简要说明该助手的核心功能。
* **操作列：** 提供查看、编辑和删除功能的入口。

## 常用操作

<img src="/assets/images/zh/admin/assistant-library-list-management-2_zh.webp" alt="" />
在列表右上角，点击 **操作 (Actions)** 按钮，展开功能菜单：

* **新建助手模板：** 手动创建一个全新的助手。
* **从模板中添加：** 访问 Cherry Studio 官方提供的预设库进行快速导入。

<Callout type="warn">
**注意：** 删除助手是不可逆操作。虽然不会影响已经将该助手保存到本地的用户，但新用户将无法再获取该助手。
</Callout>
