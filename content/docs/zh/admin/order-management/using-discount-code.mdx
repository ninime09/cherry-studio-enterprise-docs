<Callout type="info">
折扣码仅用于 **Express 套餐的购买与续费**，不适用于余额（Token）充值。Premium 版采用许可证授权，无需折扣码。
</Callout>

## 在哪里输入

下单时进入 **订单摘要** 页面，右侧即可看到「**折扣码**」输入框。

到达订单摘要页有两种方式：

- **首次开通**：注册账号 → 创建组织 → 选择套餐 → 完成企业认证后自动跳转。完整流程见 <DynamicLink href="/[lang]/docs/quickstart#express">快速开始 · Express 版</DynamicLink>。
- **续费或新增组织**：**账号中心** → **关联组织** → 点击「**创建组织**」，或对已到期组织走续费入口。

## 应用折扣码

<img src="/assets/images/zh/express/discount-checkout_zh.webp" alt="订单摘要与折扣码输入框" />

在「**折扣码**」输入框中填入折扣码（大小写不敏感），点击 **应用**。订单总额会根据应用结果更新，确认无误后即可提交订单进入支付。

## 获取折扣码

折扣码由 Cherry Studio 团队签发：

- 商务洽谈请 <DynamicLink href="/[lang]/docs/contact">联系我们</DynamicLink>。
- 营销活动期间通过官方渠道（公众号、客服）发放公开码。
