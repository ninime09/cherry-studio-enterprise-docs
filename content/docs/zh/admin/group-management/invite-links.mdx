## 功能说明

支持在每个分组内独立管理邀请链接。管理员可为不同分组创建专属的邀请链接，被邀请人通过链接注册后将自动加入对应分组。

关于被邀请人的注册流程，请参见 <DynamicLink href="/[lang]/docs/admin/user-management/invite-members">成员邀请</DynamicLink>。

## 进入邀请链接管理

1. 前往 **分组管理** 页面。
2. 点击目标分组的 **设置** 按钮，进入分组编辑页面。
3. 在左侧导航中选择 **邀请管理**。

<img src="/assets/images/zh/admin/group-management-invite-links_zh.webp" alt="邀请链接管理界面" />

邀请链接列表展示以下信息：

| 字段 | 说明 |
| --- | --- |
| **邀请码** | 系统自动生成的唯一邀请标识 |
| **状态** | **有效**（绿色）或 **已撤销**（红色） |
| **使用次数** | 已使用次数 / 最大次数（`∞` 表示不限） |
| **过期时间** | 链接到期时间，过期后将无法使用 |
| **创建时间** | 链接的创建时间 |
| **操作** | 复制链接、撤销链接 |

## 创建邀请链接

1. 点击页面右上角的 **+ 创建邀请** 按钮。
2. 在弹窗中配置以下参数：

<img src="/assets/images/zh/admin/group-management-invite-links-create_zh.webp" alt="创建邀请链接" />

| 参数 | 说明 |
| --- | --- |
| **过期时间** | 邀请链接的有效期。可选择预设时间或 **永不过期** |
| **最大使用次数** | 链接可被使用的最大次数。留空或输入 `0` 表示无限制 |

3. 点击 **创建** 完成。

## 管理已有链接

- **复制链接**：点击操作列的复制图标，将邀请链接分享给目标成员。
- **撤销链接**：点击操作列的删除图标，撤销后链接状态变为"已撤销"，已注册的用户不受影响。

<Callout type="info">
**建议：** 为不同用途创建独立的邀请链接（如"新员工入职"和"外部协作"），便于分别管理有效期和使用次数。
</Callout>
