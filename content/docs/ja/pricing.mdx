## エディション選択ガイド

### Express 版 vs Premium 版

両エディションは**コアとなるエンタープライズ機能が完全に同一**です。主な違いは導入方式と課金モデルです：

| | **Express 版**（クラウドホスト型） | **Premium 版**（プライベート導入型） |
| --- | --- | --- |
| **導入方式** | 導入不要、登録するだけですぐ利用可能 | Docker / Helm で自社導入 |
| **データ保存** | Cherry Studio クラウドで管理 | お客様のサーバーに完全保存 |
| **プロバイダー設定** | 国内外の主要 LLM を内蔵、即利用可能 | 完全自由設定、カスタムプロバイダーと私有化モデルに対応 |
| **課金方式** | サブスクリプション + Token 従量課金 | ライセンス（買い切り） |
| **ユーザー数** | プランに応じた階段制 | ライセンスに応じた階段制 |
| **技術サポート** | オンラインサポート | 専用技術サポート（年額サービス費が必要） |

### どう選ぶべき？

- **Express 版を選ぶ**：導入不要で今すぐ使いたい、主要 LLM を利用、従量課金で柔軟に
- **Premium 版を選ぶ**：データを完全に自社管理、私有化モデル連携、コンプライアンス要件がある

## Express 版の料金

### サブスクリプションプラン

Express 版は**サブスクリプション制**です。チーム規模に合わせてプランを選択：

<img src="/assets/images/ja/express/select-plan.webp" alt="Express プラン選択ページ" />

利用可能なお支払い方法：
- **Alipay** — QR コードで即時決済
- **銀行振込** — 請求書が必要な企業向け

### Token 消費

Express 版の AI 対話は Token 消費に基づく従量課金です。モデルごとに Token 単価が異なります：

<img src="/assets/images/ja/admin/provider-management-express-1_ja.webp" alt="Token 料金" />

- Token 残高はいつでもチャージ可能
- 管理コンソールで組織・メンバー別の使用量を確認

## Premium 版の料金

Premium 版は**永続ライセンス**（買い切り）で、人数に応じた階段価格です：

| チーム規模 | 説明 |
| --- | --- |
| 1〜50 人 | スタータープラン |
| 51〜200 人 | 中規模チーム |
| 200 人以上 | 大規模組織、カスタム見積もり |

<Callout type="info">
Premium 版はカスタム見積もりです。<DynamicLink href="/[lang]/docs/contact">お問い合わせ</DynamicLink> ください。
</Callout>

### 年額サポート費（任意）

Premium 版ライセンス購入後、任意で年額サポート費を追加できます：

- バージョンアップと新機能提供
- 7x24 時間の技術サポート
- リモート導入支援
- カスタム要件の相談
- 研修と利用ガイダンス

## バージョンの主要差分

| 機能項目 | Express 版 | Premium 版 |
| --- | --- | --- |
| **価格** | サブスクリプション制 | <DynamicLink href="/[lang]/docs/contact">お問い合わせ</DynamicLink> |
| **ライセンス** | サブスクリプション契約 | 商用ライセンス（買い切り） |
| **ユーザー数** | プランに応じた階段制 | ライセンスに応じた階段制 |
| **導入方式** | クラウドホスト、導入不要 | プライベート導入（Docker / K8s） |
| **技術サポート** | オンラインサポート | 専用技術サポート |
| **対応 OS** | Windows 7+, macOS, Linux | Windows 7+, macOS, Linux |
| **プロバイダー管理** | 主要 LLM を内蔵、即利用可能、**カスタム不可** | 完全自由設定、私有化モデル対応 |
| **エンタープライズ機能** | ✅ フル機能 | ✅ フル機能 |

製品の価格情報は公式サイトの比較をご参照ください：https://enterprise.cherry-ai.com/
