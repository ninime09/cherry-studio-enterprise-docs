## サブスクリプションプラン

Cherry Studio エンタープライズ版は**サブスクリプション制**です。チーム規模に合わせてプランを選択：

<img src="/assets/images/ja/express/select-plan.webp" alt="プラン選択ページ" />

利用可能なお支払い方法：
- **Alipay** — QR コードで即時決済
- **銀行振込** — 請求書が必要な企業向け

## Token 消費

AI 対話は Token 消費に基づく従量課金です。モデルごとに Token 単価が異なります：

<img src="/assets/images/ja/admin/provider-management-express-1_ja.webp" alt="Token 料金" />

- Token 残高はいつでもチャージ可能
- 管理コンソールで組織・メンバー別の使用量を確認

## コア機能

| 機能モジュール | 説明 |
| --- | --- |
| 基本機能 | マルチモデル AI 対話、マルチモーダル対応（テキスト、画像、ファイル） |
| プロバイダー管理 | 国内外の主要 LLM サービスを内蔵、すぐに使用可能 |
| 既定モデル設定 | アシスタント、トピック命名、翻訳の各モデルを設定し、体験とコストを両立 |
| ユーザー・グループ管理 | 従業員アカウントとグループ権限制御で AI リソースアクセスを精密管理 |
| アシスタント管理 | ロール・知識ベース・ツールを設定した AI アシスタントを作成し、グループごとに配布 |
| アシスタントライブラリ | 再利用可能なテンプレートを一元管理、内蔵マーケットからワンクリックインポート |
| エージェント管理 | マルチモデル連携・スキルバインド・MCP ツール統合対応の AI エージェント |
| エージェントスキル | 再利用可能なスキルモジュールを管理、手動追加またはコミュニティマーケットからダウンロード |
| 知識ベース管理 | エンタープライズ RAG エンジン、文書をベクトル化して正確な AI 回答を実現 |
| MCP サーバー | MCP プロトコルで外部ツールやデータソースを接続 |
| ミニプログラム管理 | 外部 AI ツールや Web サービスをクライアントサイドバーに統合 |
| 使用量統計 | 多次元ダッシュボードで Token 消費とコスト分布をリアルタイム監視 |
| システム設定 | ネットワークプロキシ、検索エンジン連携、OCR 文書前処理、キャッシュ管理 |
| API ゲートウェイ | API Key で外部アプリや IDE プラグインから企業モデルを呼び出し |

**対応 OS**：Windows 7+, macOS, Linux

価格情報の詳細は公式サイトをご参照ください：https://enterprise.cherry-ai.com/

## Premium 版（プライベート導入）

データを完全に自社管理したい、私有化モデルとの統合が必要、または特殊なコンプライアンス要件がある場合、Cherry Studio は Premium 版（プライベート導入）も提供しています。**永続ライセンス**（買い切り）で、人数に応じた階段価格です：

| チーム規模 | 説明 |
| --- | --- |
| 1〜50 人 | スタータープラン |
| 51〜200 人 | 中規模チーム |
| 200 人以上 | 大規模組織、カスタム見積もり |

<Callout type="info">
Premium 版はカスタム見積もりです。<DynamicLink href="/[lang]/docs/contact">お問い合わせ</DynamicLink> ください。
