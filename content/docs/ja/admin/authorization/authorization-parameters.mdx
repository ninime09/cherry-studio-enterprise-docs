**現在の License 情報** パネルにはライセンスの仕様が表示されます。これらを理解することで容量計画に役立ちます。

{/* TODO: Add screenshot for authorization parameters */}

| 項目 | 説明 | 重要ポイント |
| -------------- | --------------------------------- | ---------------------------------------------------- |
| **License ID** | 一意のライセンス識別子（UUID）。 | **サポートの照会**。Cherry Studio サポートに連絡する際はこの ID を提示してください。 |
| **有効化日時** | License を最初に取り込み有効化した日付。 | 本番稼働の開始時点を記録します。 |
| **有効期限** | License の失効日。 | **更新リマインド**。期限前にカウントダウンが表示されるため、期限前に更新してください。 |
| **状態** | 現在の法的有効性（例：`有効` / `期限切れ`）。 | システム利用可否を決定します。 |
| **会社名** | ライセンス対象の企業名。 | ライセンス主体が一致していることを監査で確認できます。 |
| **ユーザー上限** | **作成済みユーザー数 / 最大ユーザー数**（例：`6/200`）。 | **容量制御**。上限に達すると **ユーザー管理** で新規作成ができません。 |
| **ライセンス機能** | 現行版に含まれる高度機能パッケージ（例：`express`）。 | 購入した製品グレード（Standard / Enterprise / Flagship など）を示します。 |

<Callout type="warn">
**容量警告：** **ユーザー上限** が 90% に近づいたら、早めに販売チームへ連絡し枠を拡張してください。
</Callout>
