<Callout type="info" title="Premium 版のみ">
ライセンス管理は Premium 版（プライベート導入型）に適用されます。Express 版はサブスクリプション課金のため、手動でのライセンス管理は不要です。
</Callout>

**ライセンス管理** は Cherry Studio Enterprise の運用の中核です。システム管理者はこのモジュールで、合法的な利用状況の監視、ユーザー枠の管理、ライセンスの更新/移行を行います。

このモジュールは、企業 AI サービスをコンプライアンスに沿って安定稼働させ、ライセンス期限切れによるサービス停止を防ぎます。

</PremiumOnly>
