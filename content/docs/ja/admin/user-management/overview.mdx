## 主要機能エリア <a href="#heading8ca0746f50e147119c6c47d6dded5dd2-he-xin-gong-neng-qu-0" id="heading8ca0746f50e147119c6c47d6dded5dd2-he-xin-gong-neng-qu-0"></a>

**1. グローバル検索とフィルタ**

ページ上部のツールバーで効率的に検索できます：

* **ユーザー検索**：**ユーザー名**であいまい検索し対象を特定。
* **全ステータス**：
  * **アクティブ**：通常利用中のアカウント。
  * **非アクティブ**：無効化済み、または未初期化のアカウント。
* **全グループ**：所属グループ（例：「デフォルト」「開発」）で絞り込み、部門別に管理。

**2. ユーザー一覧の詳細**

テーブルに主要なユーザー情報が表示されます：
