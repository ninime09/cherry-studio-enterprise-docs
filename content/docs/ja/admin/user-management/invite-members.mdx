## 機能概要

メンバー招待では、管理者が専用の招待リンクを生成し、チームメンバーがリンクをクリックして自分で登録・組織に参加できます。管理者が 1 人ずつアカウントを作成する必要がありません。

利用シーン：
- チームの迅速な拡大
- 新入社員のセルフサービスオンボーディング
- 部門を超えたコラボレーターの追加

## 招待リンクの生成

### 操作手順

1. 管理コンソール → ユーザー管理に移動
2.「メンバーを招待」ボタンをクリック
3. 招待オプションを設定（下記パラメータ参照）
4. 生成された招待リンクをコピー

<img src="/assets/images/ja/express/invite-members.webp" alt="招待リンク生成画面" />

### 招待設定パラメータ

| パラメータ | 説明 |
| --- | --- |
| **対象グループ** | 招待された方が所属するグループ。利用可能なモデルとアシスタントが決まります |
| **有効期限** | リンクの有効期間（例：7 日、30 日、無期限） |
| **使用回数** | リンクの最大使用回数（例：無制限、10 回、50 回） |

## 招待リンクの管理

招待リンクはグループ設定内で管理します。各グループに専用の招待リンクを作成し、有効期限と使用回数を制御できます。

詳しくは <DynamicLink href="/[lang]/docs/admin/group-management/invite-links">招待リンク管理</DynamicLink> をご覧ください。

## 招待された方の登録フロー

招待リンクを受け取った方は：

1. 招待リンクをクリックして「組織に参加」ページを開く
2. 招待コードと参加するグループの情報が自動表示されます
3. 内容を確認し「今すぐ参加」をクリックして完了
4. クライアントをダウンロードしてログインすれば利用開始

<Callout type="info">
招待された方は、招待リンクで組織に参加する前にアカウントの登録とログインが必要です。クライアントのログイン方法は <DynamicLink href="/[lang]/docs/client">クライアント</DynamicLink> をご覧ください。
</Callout>

<img src="/assets/images/ja/express/invite-join.webp" alt="招待登録ページ" />

## 注意事項

- 招待リンクは適切に管理し、対象外の方に共有しないでください
- 合理的な有効期限と使用回数制限を設定することを推奨します
- 登録後も、管理者はユーザー管理でグループや権限を編集できます
- 登録済みユーザーのアクセスを取り消す場合は、ユーザー管理で「未アクティブ」状態に変更してください
