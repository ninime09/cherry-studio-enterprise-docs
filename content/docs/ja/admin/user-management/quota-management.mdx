## クォータ階層

Cherry Studio は 2 段階のクォータ体系を採用しています：

1. **グループレベルクォータ**：グループ設定で構成し、全グループメンバーのデフォルト制限として機能。
2. **ユーザーレベルクォータ**：個別ユーザーのカスタムクォータ。グループデフォルトより優先。空欄の場合はグループデフォルトを継承。

## ユーザークォータの設定

1. **ユーザー管理**ページに移動。
2. 対象ユーザーの操作列で**クォータ**ボタンをクリック。

<img src="/assets/images/ja/admin/user-management-quota-dialog-express_ja.webp" alt="ユーザークォータ設定" />

ダイアログ上部の **金額クォータ** / **トークンクォータ** タブで、金額ベースとトークンベースのクォータを切り替えて設定できます。

3. 各次元の制限値を設定：

<table><thead><tr><th width="100">次元</th><th>説明</th></tr></thead><tbody><tr><td><strong>日次</strong></td><td>過去 24 時間の最大使用量。毎日自動リセット。</td></tr><tr><td><strong>週次</strong></td><td>過去 7 日間の最大使用量。毎週自動リセット。</td></tr><tr><td><strong>月次</strong></td><td>過去 30 日間の最大使用量。毎月自動リセット。</td></tr><tr><td><strong>累計</strong></td><td>累積総使用量の上限。自動リセットなし、手動リセットが必要。</td></tr></tbody></table>

<Callout type="info">
**ヒント：** 空欄のままにするとグループデフォルトを継承します。グループにも制限がない場合は無制限となります。ダイアログ左下の **デフォルトに戻す** ボタンをクリックすると、当該ユーザーのすべてのカスタム制限を一括クリアし、グループデフォルト継承の状態に戻せます。
</Callout>

## 使用量の確認とリセット

各クォータ次元の横にある展開ボタンをクリックすると、使用量の詳細を確認できます：

<img src="/assets/images/ja/admin/user-management-quota-detail-express_ja.webp" alt="クォータ使用量詳細" />

* **現在の使用状況**：`使用量 / 制限値` の形式で当期の消費状況を表示。
* **過去 24 時間**：直近 24 時間の消費額。
* **次回リセット**：クォータが自動リセットされる時刻。
* **使用状況をリセット**ボタン：当期の使用量を手動でゼロにリセット。

## 一時クォータ加算

<Callout type="info">
追加した数量はその期間がリセットされると自動的に失効し、元のクォータ設定は変更されません。
</Callout>

あるユーザーが緊急対応などで現在のクォータを超えて作業する必要がある場合（例：月次上限に達したが、追加でバッチ処理を実行する必要があるなど）、**一時クォータ加算** を使うことで、そのユーザーの常設クォータ設定を変更せずに当期にのみ追加枠を注入できます。

### 一時クォータの追加手順

1. **ユーザー管理** に移動し、対象ユーザーの操作列で **クォータ** をクリックしてユーザークォータダイアログを開く。
2. **金額クォータ** または **トークンクォータ** タブに切り替える。
3. **1日の上限 / 週間制限 / 月間上限** の各フィールド右側（チャートボタンの隣）にある **+** ボタンをクリック。
4. 表示される `Add temporary quota — Daily/Weekly/Monthly` ポップオーバーに追加する数値（正の整数。単位は選択中のタブに従い、トークン数または通貨金額）を入力し、`Apply` をクリック。

<img src="/assets/images/ja/admin/user-management-quota-adjust-popover_ja.webp" alt="ユーザークォータ — 一時加算" />

<Callout type="warn">
追加された分は **当該期間のみ** 有効です。期間が自動リセットされるとその期間の使用量とともにクリアされ、ユーザーの常設クォータ値は変更されません。**合計限度** フィールドは一時加算をサポートしません。
</Callout>

### 加算履歴の確認

**ユーザー管理** 画面で対象ユーザー右側の **⋯** メニューを開き、`Adjustment History` を選択します。

<img src="/assets/images/ja/admin/user-management-quota-adjust-entry_ja.webp" alt="加算履歴メニュー" />

ダイアログには当該ユーザーの全期間にわたる加算履歴が時系列降順で一覧表示されます：

| 列 | 説明 |
|---|---|
| **Time** | 加算の発生時刻 |
| **Period** | 加算が適用された期間（Daily / Weekly / Monthly） |
| **Unit** | 加算の単位（Cost / Token） |
| **Amount** | 加算した数値 |
| **Operator** | 操作を実行した管理者アカウント |

<img src="/assets/images/ja/admin/user-management-quota-adjust-history_ja.webp" alt="ユーザー加算履歴" />
