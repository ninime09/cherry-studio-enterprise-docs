## 概要

Cherry Studio エンタープライズ版（Express）は、効率重視の個人とチームに **ゼロ設定** の AI ワークスペース体験を提供します。ChatGPT や Claude の登録、海外クレジットカード、特殊なネットワーク設定は不要で、世界の主要 LLM をすぐに利用できます。

## 主要特長

* **すぐ使える**：API Key やプロキシ設定は不要。
* **グローバル主要モデル**：**GPT-5**、**Claude 4.5 Sonnet**、**Google Gemini 3 Pro** などを内蔵。
* **公式価格**：課金は上流プロバイダー（OpenAI / Anthropic）と同価格。**上乗せなし**。使った分だけ支払い、残高は期限なし。
* **統一課金**：1 つの残高ですべてのモデルを利用可能。複数サブスクリプション（ChatGPT Plus、Claude Pro など）を管理する必要はありません。

## 適したユーザー

* 海外 API Key を取得できない国内ユーザー。
* ネットワーク設定をせずにすぐ作業を始めたいユーザー。
* Claude 4.5 などの高性能モデルを必要な時だけ使いたいユーザー。
