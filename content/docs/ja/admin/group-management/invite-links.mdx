## 機能概要

各グループ内で独立して招待リンクを管理できます。管理者はグループごとに専用の招待リンクを作成でき、招待された方がリンクから登録すると自動的に該当グループに参加します。

招待された方の登録フローについては <DynamicLink href="/[lang]/docs/admin/user-management/invite-members">メンバー招待</DynamicLink> をご覧ください。

## 招待リンク管理へのアクセス

1. **グループ管理**ページに移動。
2. 対象グループの**設定**ボタンをクリックして編集ページを開く。
3. 左側のナビゲーションで **Invite Links** を選択。

<img src="/assets/images/ja/admin/group-management-invite-links_ja.webp" alt="招待リンク管理画面" />

招待リンク一覧には以下の情報が表示されます：

| 項目 | 説明 |
| --- | --- |
| **招待コード** | 自動生成される一意の招待識別子 |
| **ステータス** | **有効**（緑）または **取り消し済み**（赤） |
| **使用回数** | 使用済み回数 / 最大回数（`∞` は無制限） |
| **有効期限** | リンクの有効期限。期限切れ後は使用不可 |
| **作成日時** | リンクの作成日時 |
| **操作** | リンクコピー、リンク取り消し |

## 招待リンクの作成

1. 右上の **+ Create Invite** ボタンをクリック。
2. ダイアログで以下のパラメータを設定：

<img src="/assets/images/ja/admin/group-management-invite-links-create_ja.webp" alt="招待リンクの作成" />

| パラメータ | 説明 |
| --- | --- |
| **有効期限** | 招待リンクの有効期間。プリセットまたは **永久有効** を選択 |
| **最大使用回数** | リンクの最大使用回数。空欄または `0` で無制限 |

3. **Create** をクリックして完了。

## 既存リンクの管理

- **リンクをコピー**：操作列のコピーアイコンをクリックして、招待リンクを対象メンバーに共有。
- **リンクを取り消し**：操作列の削除アイコンをクリック。取り消し後、ステータスは「取り消し済み」になりますが、既に登録済みのユーザーには影響しません。

<Callout type="info">
**ヒント：** 用途別に個別の招待リンクを作成（例：「新入社員オンボーディング」「外部コラボレーション」）すると、有効期限と使用回数を個別に管理できます。
</Callout>
