## ダッシュボード概要

コア指標ダッシュボードは、主要な KPI をカード形式で表示します。**Token** と **金額** の 2 つのビューを切り替えて確認できます。

<Tabs items={['Token', '金額']}>
<Tab value="Token">

<img src="/assets/images/ja/admin/usage-analytics-core-metrics-express-token_ja.webp" alt="Token ダッシュボード" />

### 指標の説明

| 指標 | 説明 |
| --- | --- |
| **総リクエスト数** | 選択期間内にプロバイダーへ発行された API 呼び出し成功回数。 |
| **トータルトークン** | 入力+出力トークンの合計。 |
| **入力トークン** | AI に送信されたプロンプトと文脈長。 |
| **出力トークン** | AI の返信長。 |
| **アクティブユーザー** | 選択期間内に呼び出しを行ったユニークユーザー数。 |
| **アクティブモデル** | 呼び出されたモデル数。 |
| **キャッシュ読み取り Token** | 再計算なしでキャッシュから提供されたトークン数。実際の消費を削減します。 |
| **キャッシュ書き込み Token** | 後続の再利用のためにキャッシュに書き込まれたトークン数。 |

</Tab>
<Tab value="金額">

<img src="/assets/images/ja/admin/usage-analytics-core-metrics-express-cost_ja.webp" alt="金額ダッシュボード" />

### 指標の説明

| 指標 | 説明 |
| --- | --- |
| **ライセンス総額** | 現在のライセンスの総金額枠。 |
| **ライセンス使用済み** | 消費済みのライセンス金額。 |
| **ライセンス残高** | 未使用のライセンス残額。 |
| **ライセンスリクエスト数** | ライセンスで許可されているリクエスト数の上限。 |
| **総リクエスト数** | 選択期間内に実際に発行された API 呼び出し数。 |
| **統計費用** | 選択期間内に発生した実際の費用。 |
| **アクティブユーザー** | 選択期間内に呼び出しを行ったユニークユーザー数。 |
| **アクティブモデル** | 呼び出されたモデル数。 |

</Tab>
</Tabs>

### Premium 版コア指標ダッシュボード

<img src="/assets/images/ja/admin/usage-analytics-core-metrics_ja.webp" alt="Premium 版コア指標ダッシュボード" />

#### 指標の説明

| 指標 | 説明 |
| --- | --- |
| **総リクエスト数** | プロバイダーへの API 呼び出し成功回数。 |
| **総 Token** | 入力+出力トークンの合計。コスト算定の最重要指標。 |
| **入力 Token** | AI に送信されたプロンプトと文脈長。 |
| **出力 Token** | AI の返信長。 |
| **アクティブユーザー** | 選択期間内に呼び出しを行ったユニークユーザー数。 |
| **アクティブモデル** | 呼び出されたモデル数。 |
| **中断リクエスト** | ユーザー停止/ネットワーク/タイムアウトによる失敗。 |
| **平均 Token/リクエスト** | 会話の平均複雑度。高いほど複雑/長文。 |

<Callout type="warn">
**運用ヒント:** **中断リクエスト** が増えたら **システム設定 > Model Provider** の接続状態や外部ネットワークを確認してください。
</Callout>
