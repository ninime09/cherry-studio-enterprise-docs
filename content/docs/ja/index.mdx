## 製品概要

Cherry Studio エンタープライズ版は、企業向けのクラウド型 AI ワークスペースです。導入不要、登録するだけで利用を開始できます：

- **導入不要**：アカウント登録後すぐに利用可能。サーバー準備やインフラ設定は不要
- **サブスクリプション**：チーム規模に応じたプランを選択。Alipay と銀行振込に対応
- **すぐに使えるモデル**：OpenAI、Claude、Gemini などの主要 LLM が組み込み済み。従業員が個別に設定する必要はありません

## データセキュリティと隔離

Cherry Studio エンタープライズ版はデータ保護を多層的に提供します：

| セキュリティ | 説明 |
| --- | --- |
| **データ隔離** | グループベースの厳格な隔離——各組織のデータは完全に分離され、他の組織からは参照不可 |
| **通信暗号化** | 全通信 HTTPS 暗号化 |
| **認証** | JWT + OAuth2 認証、パスワードは暗号化保存 |
| **アクセス制御** | ロールベースのきめ細かな権限体系（管理者 / グループ管理者 / メンバー） |
| **API セキュリティ** | 独立した API Key、ユーザーごとのアクセス権隔離 |

<Callout type="info" title="データ隔離メカニズム">
エンタープライズ版は共有インフラ＋アプリケーション層隔離アーキテクチャを採用しています。各組織のデータ（モデル設定、知識ベース、アシスタント、ファイルなど）はグループ隔離メカニズムにより厳密に分離されます。すべての API リクエストは認証と権限チェックを経由し、自組織のデータのみアクセス可能です。
</Callout>

| **導入方式** | 導入不要、登録するだけですぐ利用可能 | Docker / Helm で自社導入 |
